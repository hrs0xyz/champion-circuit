3 types of logins:
  1. User (player/fan)
  2. Venue (turf owner / gaming club / listing manager)  
  3. Admin (Champion Circuit staff — match control, scoring, vouchers)

A "Venue" can offer multiple Listings:
  - Himanshu Gaming Club → [Cricket pitch] [Badminton court] [PlayStation lounge]
  - Each listing = 1 activity type (physical or esports)
  - Each listing has up to 5 photos

Users play at listings → match records → points → leaderboard

Vouchers can be for anything (food, discount, free slot, merch)

Leaderboard formula = TBD later, but data model must support it

  LEADERBOARD POINTS FORMULA (for later)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Base points per match result:
  Win (ranked)     → 10 pts
  Win (casual)     → 3 pts
  Win (tournament) → 20 pts
  Draw             → 2 pts
  Loss             → 1 pt (participation)

Multipliers:
  × 1.5 if match_type = tournament
  × 1.2 if opponent rank > yours (upset win)
  × 1.0 city ranked match
  + bonus for tournament position (1st=50, 2nd=30, 3rd=15)

Admin override: score_adjustments table covers manual corrections
